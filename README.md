# Kelvin POS

## Prerequisites

To start the app locally, ensure you have the following set up:

1. **Start Docker Daemon** - Launch Docker Desktop (or your Docker daemon)
2. **Start the PostgreSQL database** - Start the PostgreSQL database as specified in `docker-compose.yaml`

You can start the database by running:
```bash
docker-compose up -d
```

## Getting Started

npm run dev
